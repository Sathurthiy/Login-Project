SECURE USER BASED AUTHENTICATION SYSTEM
               This project is based on cybersecurity which solves Bruteforce attack,SQL injection attack,Access control and session management. It uses locked feature to avoid Brute force attack where the attackers may use multiple combination of usernames and passwords to find right username and password to gain authorised access. It uses prepared statement and hashing the password using SHA256 to avoid database attack and hacking of password using usernames.It also uses session management to prevent unauthorised access to users account.
