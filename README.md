User based secure authentication
      
      The project is based on cybersecurity to prevent attacks like brute force attack ,SQL injection attack and Access control. The project uses locked feature after 3 attempts of login to avoid brute force attack where the attacker uses multiple combination of passwords and username to find the right username and password. It uses prepared statements and hashed password using SHA256 to store the password in database which prevents database attack. It also maintains a session after login to prevent unauthorised users accessing the account.
